# Installation

Get ink running on your machine in seconds.

## Install the CLI

```bash
npm install -g @ink/quill
```

Verify installation:

```bash
quill --version
```

## Run Your First Program

Create a file called `hello.ink`:

```ink
print("Hello, ink!")
```

Run it:

```bash
quill run hello.ink
```

You should see:

```
Hello, ink!
```

## Command Reference

| Command | Description |
|---------|-------------|
| `quill run <file>` | Run an ink program |
| `quill build <file>` | Build to deployable JAR |
| `quill --help` | Show all commands |

## File Naming

ink source files use the `.ink` extension:

```ink
hello.ink
myprogram.ink
game.ink
```
