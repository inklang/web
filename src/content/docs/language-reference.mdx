# Language Reference

## Variables

    let x = 10
    const PI = 3.14

## Functions

    fn greet(name) {
      return "Hello, " + name
    }

## Control Flow

    if x > 10 {
      print("big")
    } else {
      print("small")
    }

## Loops

    for i in range(10) {
      print(i)
    }
